# Next Search

## How to use

Download the example [or clone the repo](https://github.com/FacundoGFlores/nextsearch):

```sh
git clone https://github.com/FacundoGFlores/nextsearch
cd nextsearch
```

Install it and run:

```sh
npm install
npm run dev
```

## Running tests

```sh
npm test
```

## E2E tests

```sh
npm run cy:run
```
